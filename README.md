# Byte AI Coding Assistant

An advanced, secure, and context-aware coding assistant for VS Code.

## Features

- **Chat Interface**: Sidebar chat for coding assistance.
- **Context Awareness**: Reads your active file selection context automatically.
- **Command Line Access**: Can check files and run commands (with your strict permission).
- **Slash Commands**: `/explain`, `/fix`, `/doc`.
